預設變數

hyUI是由SASS語言編譯CSS而成，於SASS/_variable.scss，提供基本預設的變數，可供隨時改變色系、文字樣式以及格線系統...等設定。

注意：_variable.scss請放置最前面 import。 
                    
// 此變數為框架之整體變數，控制整體網站之設定，包含 顏色、文字、格線系統
// 於mixins 資料夾內，各個模組也有其個別使用的變數，請至個別檔案設定，個別使用變數不含在此設定

//-------------------------------------------------------------顏色---------//
// 定義顏色 此顏色可自由變換
$primaryColor:              #21BAFF;                //主色 
$secondaryColor:            #781F79;                //輔色
$lightColor:                #FCC211;                //點綴
$linkColor:                 #21BAFF;                //連結
$importantColor:            #FF3E4D;                //重要
// 但注意真實顏色與語意的關聯必須易於辨識

$color-blue:                #21BAFF;
$color-green:               #5CB85C;
$color-orange:              #F77810;
$color-red:                 #FF3E4D;
$color-yellow:              #FCC211;
$color-purple:              #AA5AC5;
$color-gray:                #CCC;
$color-dark:                #333;
$color-hr:                  #666;                    //hr顏色
$color-word:                #222;                    //一般字的顏色
// 連結顏色
$aColor:                    #21BAFF;
$aHover:                    lighten($aColor,10);

// 表單focus顏色
$form-focus:                #21BAFF;                //表單
//-------------------------------------------------------------字型--------//

// 字型設定
$font-family:               Lato, "PingFang TC", "Helvetica Neue", Helvetica, 微軟正黑體, Arial, sans-serif;
$font-size:                 1em;

//-------------------------------------------------------------格線系統-----//

// 格線系統
$grid-columns:              12 !default;               //格線欄位數
$grid-gutter-width:         30px !default;             //左右padding
$container-max:             1200px !default;           //container最大寬

//mediaquery
$screen-lg:                 1400px !default;           //電腦
$screen-md:                 992px  !default;           //平板
$screen-sm:                 768px  !default;           //手機
$screen-xs:                 576px  !default;           //極小尺寸

//-------------------------------------------------------------客製化------//

$rounded-angle:             10px;                      //整站導角
$fatfooter-length:          6;                         //fatfooter選單一行個數

自由變化的色彩計畫

使用有系統的色彩計畫，可以替網站帶來舒適感及整體感。hyUI提供基本的色彩變數，對應色彩計畫，可迅速為網站迅速變換色系。下圖是hyUI預設的色彩計畫。

以色彩名稱定義色彩

因應不同的需求，不以『語意』定義顏色，而是以『色相』定義，這樣可方便延伸色彩的運用，也可後續自訂語意。以下色彩為hyUI定義之預設色彩及色碼。

  • BLUE#21BAFF
  • GREEN#5CB85C
  • ORANGE#F77810
  • RED#FF3E4D
  • YELLOW#FCC211
  • PURPLE#AA5AC5
  • DARK#333333
  • GRAY#CCCCCC

一體適用的字型

                    
$font-family:               Lato, "PingFang TC", "Helvetica Neue", Helvetica, 微軟正黑體, Arial, sans-serif;
$font-size:                 1em;

提供網站預設匯入之字型，如需要參考字型設定，可瀏覽typography單元

Grid System格線系統

                    
// 格線系統
$grid-columns:              12 !default;               //格線欄位數
$grid-gutter-width:         30px !default;             //左右padding
$container-max:             1200px !default;           //container最大寬

//mediaquery
$screen-lg:                 1400px !default;           //電腦
$screen-md:                 992px  !default;           //平板
$screen-sm:                 768px  !default;           //手機
$screen-xs:                 576px  !default;           //極小尺寸

格線系統

因應相容於舊版IE瀏覽器(含IE9以下)，不考慮用flexbox的CSS技術，目前hyUI格線系統以Bootstrap 3為原型，變數預設提供可變動之grid-column、grid-gutter-width之預設設定，皆參照Bootstrap 12欄位數以及左右padding各15px為預設值，當然只要更改預設值便可以建立自己的格線系統，注意grid-gutter-width為左右padding加總，數值將平均分配左右兩側。

mediaquery

mediaquery之預設值，以Bootstrap 4的斷點做為分界，以1400px、992px、768px、576px分作為5個響應式斷點。

相關詳細設定可參考grid system以及mediaquery

自訂客製化變數

                    
//-------------------------------------------------------------客製化------//

$rounded-angle:             10px;                      //整站導角
$fatfooter-length:          6;                         //fatfooter選單一行個數

hyUI可自訂自己的SASS變數，可寫在最下方，只要記得勿覆蓋原生的變數名稱即可。
