從GitHub網站下載完整的hyUI kit原始程式碼。

GitHub 檔案下載

下載之後您會得到以下的檔案

  • html檔案
  • js資料夾
  • SASS資料夾
  • CSS資料夾
  • images資料夾
  • vendor資料夾
  • config.rb
本框架需要具有編譯SASS/SCSS能力的開發人員使用。

檔案結構

                    
├── sass/
│   ├── page/
│   │   ├── _template.scss              // template
│   │   ├── _mp.scss                    // mp
│   │   ├── _cp.scss                    // cp
│   │   ├── _lp.scss                    // lp
│   │   ├── _np.scss                    // np
│   │   ├── _qp.scss                    // qp
│   │   ├── _fp.scss                    // fp
│   │   ├── _sp.scss                    // sp
│   │   └── _sitemap.scss               // sitemap
│   ├── sass/
│   │   ├── mixins/
│   │   └── ...                         // scss所有檔案
│   ├── _variable.scss                  // 變數
│   └── hyui.scss                       // 統一匯出之主要SCSS
│
├── css/
│   └── hyui.css                        // sass輸出檔案
│
├── js/
│   ├── jquery.3.3.1.min.js             // Jquery 3v.
│   ├── jquery.1.12.4.js                // Jquery 1v.
│   ├── html5shiv.js                    // html5shiv
│   ├── respond.min.js                  // respond
│   ├── hyui.js                         // hyuijs   *勿更動
│   ├── hyui.min.js                     // 壓縮hyuijs   *勿更動
│   └── customize.js                    // 客製化js
│
├── vendor/
│   ├── jquery.easing.min.js
│   └── ...                             //自行客製 或 載入plugin(ex:slick)
│
└─ images/
    ├── basic/                          //基本預設圖檔
    ├── icon/                           //hyUI icon(svg)
    └── ...                             //專案所需圖檔
