隨時自由建構自己常用的Mixin

HyUI提供常用之mixin設定，一方面套用預設變數快速改變介面的樣貌，另一方面則是處理css hack的修正，其中最多的是前綴prefix以修正不同瀏覽器之間的相容性問題。

mixin將由sass/_mixin.scss匯入資料夾mixins裡所有的預設mixin檔案。

常用Mixin

                        
@include make_col(2, 1);

HyUI Mixin

                        
//  MIXIN
@import "mixins/_clearfix";                 //清除浮動
@import "mixins/_mediaquery";               //mediaquery
@import "mixins/_img-responsive";           //圖片RWD設定
@import "mixins/_transition";               //動畫
@import "mixins/_opacity";                  //透明度
@import "mixins/_box-sizing";               //box-sizing
@import "mixins/_border-radius";            //導角
@import "mixins/_gradient";                 //漸層
@import "mixins/_center-block";             //置中
@import "mixins/_text-overflow";            //ellipsis 字多點點
@import "mixins/_box-shadow";               //陰影
@import "mixins/_li-reset";                 //清除li格式
@import "mixins/_bootstrap_grid";           //格線系統
// 客製化
@import "mixins/_extend";                   //客製化繼承
@import "mixins/_fatfooter";                //fatfooter
@import "mixins/_image";                    //img設定

SASS設定範例

                        
// mixin設定
@mixin border-radius($radius) {
    -webkit-border-radius: $radius;
    border-radius: $radius;
}
// scss設定
@include border-radius(15px);

css輸出已經預設加入prefix前綴詞


// CSS輸出
-webkit-border-radius: 15px;
border-radius: 15px;

擴充套件：支持Compass 與 Bourbon

Compass是一套非常優秀的Sass擴充套件，Compass目前已經停止更新，但以完整度來說是足夠支持hyUI所需要的功能。

Bourbon是一套更簡潔、輕量級的SASS函式庫。

hyUI目前預設使用了Compass Vertical Rhythm的設定

在sass的擴充套件裡，垂直的文字韻律可以讓使用者瀏覽網站有更佳的閱讀體驗，因此hyUI在文章設定已經預先做了垂直韻律的排版，參考資料：https://drewish.com/tools/vertical-rhythm/

詳細設定請參照typography

                        
@import "compass";
$base-font-size: 16px;
$base-line-height: 16px;
$rhythm-unit: 'em';
$base-font-stack:$font-family;
@include establish-baseline();
$grid-background-baseline-height:2em;
$show-baseline-grid-backgrounds: true;
$grid-background-baseline-color: rgba(187, 187, 187, 0.4);
