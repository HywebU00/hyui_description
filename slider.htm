Slider

檔案名稱：sass/_slick.scss

HyUI使用slick的輪播模組，目前hyUI vendor內含slick已經有加入客製化設定，如需要最新版，請到slick官網下載。

請在網頁中匯入slick 外掛

slick屬於外掛，故放在vendor資料夾裡，包括所有的圖檔、js、css

匯入外掛CSS

                
<link rel="stylesheet" type="text/css" href="vendor/slick/slick.css"/>
<link rel="stylesheet" type="text/css" href="vendor/slick/slick-theme.css"/>

匯入JS

                
<script type="text/javascript" src="vendor/slick/slick.min.js"></script>

Slick的基本設定

使用設定以slick官方文件為準。

                
$(document).ready(function(){
  $('.your-class').slick({
    arrows: true,                       //左右箭頭
    autoplay: false,                    //自動播放
    autoplaySpeed: 3000,                //自動播放秒數
    dots: false,                        //顯示圓點
    dotsClass:  'slick-dots',           //原點css
    draggable: true,                    //滑鼠可以拖曳
    infinite: true,                     //無限輪播
    pauseOnHover: true,                 //滑鼠移過後暫停自動撥放
    pauseOnDotsHover: false,            //滑鼠移過圓點後暫停自動撥放
    rtl: false,                         //改變輪播方向
    slidesToShow: 1,                    //一次顯示幾張
    slidesToScroll: 1,                  //一次輪播幾張
    vertical: false                     //改成垂直方向
  });
});

SCSS設定

                
// slick._scss
// 首頁大圖輪播
.mpSlider {
    width: 100%;
    padding-bottom: 3em;
    @include gutter(0);
    .img-container {
        @include aspect-ratio(5, 2); //PC版比例
        @include screen('mobile') {
            @include aspect-ratio(5, 3); //Mobile版比例
        }
    }
    .slick-prev {
        @include screen('mobile') {
            top: 35%;
            left: 2%;
        }
    }
    .slick-next {
        @include screen('mobile') {
            top: 35%;
            right: 2%;
        }
    }
}

JQuery設定

關於slick按鈕的title語系

因應無障礙需求，在左右箭頭需要有替代文字，但往往因為網站有不同語系需求，因此按鈕需要呈現不同語言的替代文字，例如
<html lang="zh-Hant" > 中按鈕會呈現 <button class="slick-prev" ... title="前一則">Previous</button> 或是
<html lang="en" > 中按鈕會呈現或是 <button class="slick-prev" ... title="previous">Previous</button> ，
此時無法用同一種語言因應各種語系，HyUI提供一組判斷語系之javascript設定，語系參照W3C ISO Language Codes，可自行複製到各專案修改語系及對應之替代文字即可。注意：此程式並無預設放入hyui.js。

其中，程式是判斷語系字串中的前面字符作為判斷依據，目的是同一語系可能會有分支語系，但以開頭之語系英文字母做判斷。

範例：

                
//不同語系
//無障礙切換slick箭頭語系
if ($('html')[0].hasAttribute("labg")) {
    var weblang = $('html').attr('lang');
    //如果是中文語系
    if (weblang.substring(0, 2) == 'zh') {
        $('.slick-prev').attr('title', '上一筆');
        $('.slick-next').attr('title', '下一筆');
    //如果是非中文語系
    } else if (weblang.substring(0, 2) !== 'zh') {
        $('.slick-prev').attr('title', 'previous');
        $('.slick-next').attr('title', 'next');
    }
}
