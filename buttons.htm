Buttons / 按鈕

檔案名稱：sass/_btn.scss

以顏色定義按鈕

按鈕可集合在父層可選擇設定.btn_grp包成一個按鈕區。按鈕會呈現inline-block橫排置中呈現。

按鈕 按鈕 按鈕 按鈕 按鈕 按鈕 按鈕 
                
<div class="btn_grp">
    <a class="btn">...</a>
    <a class="btn btn-blue">...</a>
    <a class="btn btn-green">...</a>
    <a class="btn btn-orange">...</a>
    <a class="btn btn-yellow">...</a>
    <a class="btn btn-purple">...</a>
    <a class="btn btn-red">...</a>
</div>

不同tag皆可套用，但請注意html語意須正確

a,button, 或是input

按鈕 
                
<div class="btn_grp">
    <a href="#" class="btn" role="button">按鈕</a>
    <button type="button" class="btn">按鈕</button>
    <input type="button" class="btn" value="Input 按鈕">
    <input type="submit" class="btn" value="Submit 按鈕">
</div>
