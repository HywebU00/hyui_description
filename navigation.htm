Navigation 導覽列

檔案名稱：sass/_header.scss
檔案名稱：sass/_menu.scss

導覽列由nav.navigaton包覆，預設內含導覽連結，可設定兩組，如只有一組則預設靠右，如有兩組將會放置於左右兩側；如偵測到有放置語言選單，導覽列右側會自動產生空間放置選單，以上設定皆已於hyui.js判斷。

HTML範本

                
<!-- navigation Start -->
<nav class="navigation">
    <!-- 一組ul預設靠右 -->
    <!-- 兩組ul已寫JQ靠左靠右 -->
    <ul>
        <li><a href="#">回首頁</a></li>
        <li><a href="#">網站導覽</a></li>
        <li><a href="#">English</a></li>
        <li><a href="#">常見問答</a></li>
    </ul>
    <ul>
        <li><a href="#">第二組導覽</a></li>
        <li><a href="#">第二組導覽</a></li>
        <li><a href="#">第二組導覽</a></li>
    </ul>
    <div class="language">
        <a href="#">語言選擇</a>
        <ul>
            <li><a href="#">繁體中文</a></li>
            <li><a href="#">简体中文</a></li>
            <li><a href="#">ENGLISH</a></li>
        </ul>
    </div>
</nav>
<!-- navigation End -->

JQuery設定

                
/*-----------------------------------*/
//////////// nav如果有兩個選單///////////
/*-----------------------------------*/
var _navLength = $('.navigation ul').length;
if (_navLength > 1) {
    $('.navigation ul:nth-child(1)').addClass('left_nav');
}
$('.navigation').has('.language').addClass('have_language');
