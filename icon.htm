Icons / 圖示

檔案名稱：sass/_icons.scss

hyUI提供一套簡單快速套用的icon圖示，與@font face差別在於，本圖示乃是用svg圖呈現，並非以網路字型技術呈現。

一般樣式

                
<ul>
    <li class="i_apple"></li>
    <li class="i_arrow_left"></li>
    <li class="i_arrow_right"></li>
    <li class="i_bookmark"></li>
    <li class="i_chat"></li>
    <li class="i_check"></li>
    <li class="i_clock"></li>
    <li class="i_close"></li>
    <li class="i_edit"></li>
    <li class="i_facebook"></li>
    <li class="i_googleplus"></li>
    <li class="i_grid"></li>
    <li class="i_heart"></li>
    <li class="i_home"></li>
    <li class="i_info"></li>
    <li class="i_link"></li>
    <li class="i_linkedin"></li>
    <li class="i_lock"></li>

</ul>

</button>

</button>
