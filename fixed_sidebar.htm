Fixed_sidebar / 固定側邊欄

檔案名稱：sass/_fixed_sidebar.scss

hyUI提供一個固定側邊欄的作法，固定特定位置，而不隨滾動欄滾動。

範例樣式

  • Ａ
  • Ｂ
  • Ｃ
  • Ｄ 
                
<!-- fixed_sidebar start -->
<section class="fixed_sidebar">
    <ul>
        <li><a href="#" title="指定區塊名稱1" id="goto_A">Ａ</a></li>
        <li><a href="#" title="指定區塊名稱2" id="goto_B">Ｂ</a></li>
        <li><a href="#" title="指定區塊名稱3" id="">Ｃ</a> </li>
        <li><a href="#" title="指定區塊名稱4" id="">Ｄ</a> </li>
    </ul>
</section>
