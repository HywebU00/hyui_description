Typography / 文字設定

檔案名稱：sass/_typography.scss

字型基本設定

                
// 字型設定
$font-family:               Lato, "PingFang TC", "Helvetica Neue", Helvetica, 微軟正黑體, Arial, sans-serif;
$font-size:                 16px;

Verticla Rhythm設定，打造閱讀最佳體驗

                
@import "compass";
$base-font-size: 16px;
$base-line-height: 16px;
$rhythm-unit: 'rem';
$base-font-stack:$font-family;
// +leader() = 設定margin-top
// +padding-leader() = 設定padding-top
// +trailer() = 設定margin-bottom
// +padding-trailer() = 設定padding-bottom
.cp {
    @include adjust-font-size-to($base-font-size*1, 2); //預設cp的文字設定(有可能上搞資料未包標籤)
    h1 {
        @include adjust-font-size-to($base-font-size*2, 3);
        @include padding-leader(0);
        @include trailer(.5);
    }
    h2 {
        @include adjust-font-size-to($base-font-size*1.5, 2);
        @include padding-leader(0);
        @include trailer(0);
    }
    h3 {
        @include adjust-font-size-to($base-font-size*1.25, 2);
        @include padding-leader(0);
        @include trailer(0);
    }
    h4 {
        @include adjust-font-size-to($base-font-size*1.25, 2);
        @include padding-leader(0);
        @include trailer(0);
    }
    h5 {
        @include adjust-font-size-to($base-font-size*1.25, 2);
        @include leader(0);
        @include trailer(0);
    }
    h6 {
        @include adjust-font-size-to($base-font-size*1, 2);
        @include leader(0);
        @include trailer(0);
    }
    p {
        @include adjust-font-size-to($base-font-size*1, 2);
        @include leader(0);
        @include trailer(2);
    }
    ul,
    ol {
        @include adjust-font-size-to($base-font-size*1, 2);
        @include leader(0);
        @include trailer(2);
    }
    dd,
    th,
    td figure {
        @include adjust-font-size-to($base-font-size);
    }
    blockquote {
        background: #f2f2f2;
        font-style: normal;
        display: block;
        @include adjust-font-size-to($base-font-size);
        border-left: 5px solid $primaryColor;
        padding: $base-font-size*2;
        box-sizing: border-box;
        margin-left: .5em;
        margin-right: .5em;
        @include leader(0);
        @include trailer(2);
    }
    img {
        max-width: 100%;
        @include screen('mobile') {
            max-width: 100%;
            width: auto;
            clear: both;
        }
    }
}

  • 打造有規律的閱讀節奏
